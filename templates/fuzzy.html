Fuzzy Logic Control

  • My third year independent project was researching the traction control subsystem of an 800V electric vehicle.
  • This research involved modelling the EV powertrain using Simulink, and then designing a controller to modulate the output torque of the electric motor.
  • Fuzzy Logic Control is a popular method to reduce torque ripples within the motor, and the results showed that this was successful.
  • The implementation of FLC can be complex as it involves programming the controller behaviour using rules.
  • Utilising FLC helped me to learn how to integrate a codebase into an existing system.
