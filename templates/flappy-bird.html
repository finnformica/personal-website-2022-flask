Flappy Bird + NEAT

  • The Flappy Bird game utilises pygame to handle the screen rendering and player movement.
  • The genetic algorithm, NEAT, involves evolving artificial neural networks to improve the performance of later generations.
  • The best performing neural networks of each "species" are mutated and play the game again.
  • Due to the simplicity of Flappy Bird, a very good neural network evolves very quickly.
  • From this project I learnt a lot about how Neural Networks function under the hood and how to run multiple instances of a NN in a game.
